Profil Singkat

Saya adalah anak pertama dan satu satunya dalam keluarga saya. Saya berasal dari keluarga sederhana dengan berjuta kebahagiaan. Saya mempunyai cita cita menjadi seorang milyuner tetapi itu sangatlah sulit untuk digapai tetapi saya tidak akan menyerah untuk menggapai cita cita saya.

Dari kecil saya mempunyai cita cita untuk menjadi seeorang perwira Tentara Nasional Indonesia. Saya bermimpi agar saya dapat diterima di Akademi Militer tetapi itu sangat tidak mungkin karena Akademi Militer tidak menerima lulusan SMK. Tetapi itu mematahkan semangat saya dalam menggapai impian saya menjadi seorang Perwira TNI.
